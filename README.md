# PARANOIA

> **YOU ARE BEING OBSERVED.**

First-person psychological horror game by **vamp9**. Inspired by *Outlast 2*.

You wake on a dark forest path. Your only tool is a smartphone — its flashlight,
its camera, its battery. It is always recording. Something out there is always
watching. There are no weapons. You cannot fight. You can only move, hide,
manage your light, and decide — at the very end — whether to trust the thing
in your hand.

## 🎮 Download & Play

Grab **`PARANOIA-v1.0.0-win64.exe`** from
[**Releases**](https://github.com/lordvamp9/paranoia-game/releases) — a single
portable executable, no installation required. Double-click and play.

> 🎧 **Headphones strongly recommended.** The audio is 3D-positional — you will
> hear them before you see them.

| Input | Action |
|---|---|
| `WASD` | Move |
| Mouse | Look (smartphone POV) |
| `Shift` | Sprint (they hear it) |
| `F` | Flashlight on/off |
| `E` | Interact |
| `0-9` | Enter codes |
| `F11` | Toggle fullscreen |
| `Esc` | Release mouse |

**Campaign:** single run, ~20–30 minutes. Two endings.

## 🌑 What's inside

- **Smartphone Locked POV** — the phone is your camera, your HUD (battery %,
  objective, REC) and your only light. The screen glitches and the text
  corrupts as your stress rises.
- **Shadow Entities** — pure-black humanoids with a five-state AI
  (patrol → alerted → pursuit → coordinated hunt → retreat). They react to
  light, proximity and sound; multiple entities flank and cut your escape.
- **Stress system** — chromatic aberration, film grain, glitch blocks,
  heartbeat and pitch-shifted audio scale with how hunted you are.
- **Battery management** — the light burns ~20 min of battery; pressure from
  nearby entities drains it faster. At 0% the screen dies.
- **3 phases** — the path (Awakening), the liminal forest (fountain, coded
  cabin, crossroad sign), and the white house (kitchen key, mirror puzzle,
  basement, hidden room).
- **4 puzzles** — Water Key, Door Code 2-7-4-1, Mirror Reflection, Final Truth.
- **2 endings** — *TRUST ME* / *REJECT TRUTH*.
- **Everything procedural** — every model, texture and sound is generated by
  code. No external assets.

## 🛠 Tech

Originally specced for UE5/C++; this implementation ships the same design on a
lighter stack so it can build and run anywhere:

- **Engine:** [Three.js](https://threejs.org/) (WebGL2) — custom post-processing,
  instanced low-poly forest, dynamic shadow-mapped flashlight
- **Audio:** WebAudio — fully synthesized (wind, whispers, entity breathing,
  heartbeat), HRTF 3D panning, stress-coupled distortion
- **Shell:** Electron → single portable Windows x64 `.exe`

### Build from source

```bash
git clone https://github.com/lordvamp9/paranoia-game.git
cd paranoia-game
npm install
npm start          # run in dev
npm run dist       # package portable .exe into dist/
```
