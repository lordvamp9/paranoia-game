# Changelog

All notable releases of **PARANOIA**, a first-person psychological horror
game by **vamp9**.

## v4.1.0 — Definitive native build

The canonical release. Native **C++17 + raylib** (OpenGL): custom dual-resolution
renderer, real-time software-synthesized audio, dynamic flashlight shadow
mapping. Ships as a **single self-contained Windows executable** (~2.8 MB) — no
installer, no external libraries, no loose files. Double-click and play.

This build supersedes and retires the experimental v4.0.0 Godot port.

## v4.0.0 — Godot 4 port *(retired)*

An experimental re-implementation in Godot 4 (C++ GDExtension + GDScript). It
regressed on visual fidelity and shipped as a bulky multi-file package (an
~87 MB executable plus a separate native DLL). Retired in favour of the native
raylib build it was derived from, which remains superior in every respect.

## v3.8.0 — Display mode

Settings: borderless-window / fullscreen toggle (`F11` in-game).

## v3.1.0 — World coherence

Dual-resolution render, humanoid spectres, phone charging, world-coherence pass.

## v3.0.0 — The full game

Menus, cinematic intro, settings, full narrative and puzzle rework.

## v2.0.0 — Native rewrite

Full rewrite from the web prototype to native C++ / raylib.

## v1.0.0 — First release

Original first-person horror prototype (Three.js + Electron).
